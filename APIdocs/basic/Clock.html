                          # Clock

                          该对象用于跟踪时间。如果performance.nowopen in new window可用，则 Clock 对象通过该方法实现，否则回落到使用略欠精准的Date.nowopen in new window来实现。

                          # 构造函数

                          # Clock( autoStart : Boolean )

                          autoStart — (可选) 是否要在第一次调用 .getDeltaopen in new window() 时自动开启时钟。默认值是 true。

                          # 属性

                          # .autoStartopen in new window : Boolean

                          如果设置为 true，则在第一次调用 .getDeltaopen in new window() 时开启时钟。默认值是 true。

                          # .startTimeopen in new window : Float

                          存储时钟最后一次调用 startopen in new window 方法的时间。默认值是 0。

                          # .oldTimeopen in new window : Float

                          存储时钟最后一次调用 startopen in new window, .getElapsedTimeopen in new window() 或 .getDeltaopen in new window() 方法的时间。默认值是 0。

                          # .elapsedTimeopen in new window : Float

                          保存时钟运行的总时长。默认值是 0。

                          # .runningopen in new window : Boolean

                          判断时钟是否在运行。默认值是 false。

                          # 方法

                          # .startopen in new window () : undefined

                          启动时钟。同时将 startTimeopen in new window 和 oldTimeopen in new window 设置为当前时间。 设置 elapsedTimeopen in new window 为 0，并且设置 runningopen in new window 为 true.

                          # .stopopen in new window () : undefined

                          停止时钟。同时将 oldTimeopen in new window 设置为当前时间。

                          # .getElapsedTimeopen in new window () : Float

                          获取自时钟启动后的秒数，同时将 .oldTimeopen in new window 设置为当前时间。 如果 .autoStartopen in new window 设置为 true 且时钟并未运行，则该方法同时启动时钟。

                          # .getDeltaopen in new window () : Float

                          获取自 .oldTimeopen in new window 设置后到当前的秒数。 同时将 .oldTimeopen in new window 设置为当前时间。 如果 .autoStartopen in new window 设置为 true 且时钟并未运行，则该方法同时启动时钟。
