# 任意几何图形外观(MaterialAppearance)

支持材质着色的任意几何图形的外观。

## 构造器

**MaterialAppearance(options)**

options具有以下属性的对象：

| Name     | Type | Default | Description                        |
| :------- | :--- | :------ | :--------------------------------- |
| material |      |         | `optional`用于确定片元颜色的材质。 |

## 示例代码

```javascript
new Webgis.MaterialAppearance({
      material: labelMaterial,
    }),
```
