# 多边形(PolygonMaterial)

多边形材质部分。具有透明度渐变效果。

## 构造器

**PolygonMaterial(color,height)**

| Name   | Type                      | Description        |
| :----- | :------------------------ | :----------------- |
| color  | [Color](../math/Color.md) | 多边形颜色。       |
| height | number                    | `option`多边形高度 |

## 示例代码

```javascript
const polygonMaterial = new Webgis.PolygonMaterial(new Webgis.Color(184, 189, 51,0.5),50); 
```
