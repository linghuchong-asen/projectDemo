# 墙(WallMaterial)

墙材质部分。具有流体墙效果。

## 构造器

**WallMaterial(color)**

| Name  | Type                      | Description  |
| :---- | :------------------------ | :----------- |
| color | [Color](../math/Color.md) | 多边形颜色。 |

## 示例代码

```javascript
const wallMaterialFluid = new Webgis.WallMaterial(new Webgis.Color(60, 158, 78,0.6));
```

## 属性

### isFluid: boolean

是否开启流体墙特效。
