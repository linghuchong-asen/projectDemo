# 文字(LabelGeometry)

文字几何体部分。

## 构造器

**LabelGeometry()**

## 示例代码

```javascript
const labelGeometry = new Webgis.LabelGeometry();
```

## 属性

### position: [Vector3](../math/Vector3.md)

获取或设置文字的位置。

### scale: [Vector3](../math/Vector3.md)

获取或设置文字的缩放比例。

### offset: [Vector3](../math/Vector3.md)

获取或设置文字的偏移量。
